OrderGoodsClient
================
Демонстрационное Android-приложение.

Перед началом
-------------

Приложение использует REST-сервис og-service-public.

Используемые библиотеки
-----------------------

* Foundation
  * Android KTX
  * Test
* Architecture
  * Data Binding
  * LiveData
  * Navigation
  * Room
  * ViewModel
* UI
  * Fragment
  * Layout
* HTTP client
  * Retrofit2
    * OkHttp3
    * Network Interceptors
* Third party and miscellaneous libraries
  * JavaRX2
  * Dagger2 + MVP
  * Kotlin Coroutines
